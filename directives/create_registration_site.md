# Directive: Registration Website for Instituto Nadar

## Goal
Create a modern, premium registration website for Instituto Nadar with three main areas: Main/Registration Page, Admin Panel, and Student Area.

## Requirements
- **Main Page**:
    - Top banner.
    - "Faça sua Matrícula" (Make your registration) button.
    - Navigation menu to access Admin and Student areas.
- **Admin Panel**: Placeholder/Initial setup for administrative tasks.
- **Student Area**: Placeholder/Initial setup for student access.
- **Design**: Premium aesthetics, high-fidelity UI, responsive layout.
- **Tech Stack**: HTML, Vanilla CSS, Javascript.

## Steps
1. **Design System**: Define a premium color palette (e.g., Deep Sea Blue, Gold/Sand accents) and typography.
2. **Base Styles**: Create `index.css` with core tokens and utilities.
3. **Main Page (`index.html`)**:
    - Header with logo and navigation menu.
    - Hero section with the top banner and "Faça sua Matrícula" button.
    - Responsive layout.
4. **Admin Page (`admin.html`)**: Basic structure for the admin dashboard.
5. **Student Page (`aluno.html`)**: Basic structure for the student area.
